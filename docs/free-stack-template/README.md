# Free Stack Template

도메인 갱신비 외 운영비 **0원**으로 한국어 웹 서비스를 띄우는 템플릿.

- 👉 **[PLAYBOOK.md](./PLAYBOOK.md)** — 사람이 읽고 따라하는 매뉴얼 (아키텍처/단계별 셋업/함정).
- 👉 **[LLM_BOOTSTRAP_PROMPT.md](./LLM_BOOTSTRAP_PROMPT.md)** — LLM 에 통째로 던지는 프롬프트 템플릿.

## 스택

Next.js 16 (App Router) + React 19 + TypeScript + Supabase (Postgres/Realtime) + Cloudflare Workers (`@opennextjs/cloudflare`) + GitHub.

## 빠른 시작

1. PLAYBOOK 의 §1 사전 준비 → §4 까지 사람이 직접 따라 한다, 또는
2. LLM_BOOTSTRAP_PROMPT 의 빈칸을 채워 LLM 에 던지고 검수만 한다.

둘 다 결과물은 같다.
